## Usage

### Interactive Shell
```
[Ja4instaHome]> login myuser mypassword
[Ja4instaHome]> target username
[Ja4instaHome]> info
[Ja4instaHome]> engagement
[Ja4instaHome]> shadowban
[Ja4instaHome]> followers 500
[Ja4instaHome]> posts 50
[Ja4instaHome]> track 60
[Ja4instaHome]> report
```

### Ja4BAN Workflow
```
# Step 1 — Create sessions
[Ja4instaHome]> create 20

# Step 2 — Load sessions
[Ja4instaHome]> ban sessions ACC_SESSIONS_IG.txt

# Step 3 — Report target
[Ja4instaHome]> ban targetuser spam
```

### CLI Mode
```bash
# Account info
python Ja4instaHome.py -u target -info -engagement -activity

# Download everything
python Ja4instaHome.py -u target -posts -stories -highlights -reels -propic

# Ja4BAN via CLI
python Ja4instaHome.py --ban targetuser --reason spam --sessions mysessions.txt --rps 5

# Create accounts
python Ja4instaHome.py --create 20

# Monitor changes every 60s
python Ja4instaHome.py -u target --track --interval 60
```

## Command Reference

### Ja4BAN
| Command | Description |
|---|---|
| `ban <user> [reason]` | Report account — reason: spam / scam / drugs / nudity / violence / hate |
| `ban sessions <file>` | Load session IDs from a text file (one per line) |
| `bansessions [file]` | Show loaded sessions or load from file |
| `create [n]` | Create n Instagram accounts, saves sessions to `ACC_SESSIONS_IG.txt` |

### Information
| Command | Description |
|---|---|
| `info` | Full account info — ID, bio, followers, phone, email |
| `engagement` | Engagement rate with benchmark |
| `activity` | Hourly & daily posting chart |
| `shadowban` | 3-layer shadowban detection |
| `agecheck` | Estimate account age from oldest post |
| `toppost` | Top 10 posts by likes |

### Social Graph
| Command | Description |
|---|---|
| `followers [n]` | Fetch up to N followers, saved to file |
| `following [n]` | Fetch up to N following, saved to file |
| `mutual <user2>` | Mutual followers between two accounts |
| `ghost` | Detect ghost / inactive followers |

### Downloads
| Command | Description |
|---|---|
| `posts [n]` | Download posts with progress bar (0 = all) |
| `stories` | Download active stories |
| `highlights` | Download all highlights albums |
| `reels [n]` | Download reels |
| `propic` | Full-resolution profile picture |

### OSINT
| Command | Description |
|---|---|
| `emails` | Extract emails from bio + scan followers |
| `phones` | Phone numbers from bio + Instagram lookup |
| `dorks` | Google dork queries for the target |
| `geo [n]` | Geo location data from posts |

### Export
| Command | Description |
|---|---|
| `export json` | Export all session results to JSON |
| `export csv` | Export account info to CSV |
| `report` | Generate full HTML report |
